
# 🃏 MTG Card manager for Obsidian Base

Simple python script to manage an Obsidian Base of MTG cards

## 🎉 Currently Features

- Create a new card file from Scryfall URL with:
    - Card Image
    - Collection
    - Card Type
    - Card Number
    - Grouping Tag
    - Additional information
- Delete a card from base and removing the attached image
- Remove unused image files
- Grouping Tags
    - List all tags
    - Create
    - Edit
    - Delete
